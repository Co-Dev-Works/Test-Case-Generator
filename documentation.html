What this tool does

Paste a requirement (natural language). The generator classifies the requirement (login, signup, upload, payment, etc.), extracts simple constraints (file size, formats, digits), and produces structured test cases (title, steps, expected, priority, tags).

Fields explained

  • Title: short descriptive title of the test case.
  • Steps: numbered steps to reproduce the test.
  • Expected: expected result/behavior.
  • Priority: High / Medium / Low (based on domain & keywords).
  • Tags: categories like positive, negative, security, validation.
  • Sample data: generated sample inputs (emails, OTPs, filenames).

Exports

You can download generated test cases as CSV, XLSX, or JSON using the buttons on the results page.

Tips

  • Add explicit constraints in the requirement (e.g., "up to 5MB", "only JPG/PNG", "6-digit OTP") to get better tailored cases.
  • Use the dashboard to check coverage and priority distribution.
